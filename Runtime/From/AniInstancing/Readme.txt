Baking:
	- Put prefab into the scene Prefabs/[Bake]Singleton
	- Navigate to the top bar and press AnimationInstancing -> Animation Generator
	- Asset to Generate - Choose prefab from the assets. Prefab should have the Animator component with the right AnimationController
	- Optionally - enable attachments, add FBX model of the item that will be attached
	- Choose which animations to bake
	- Press generate.
(Play Mode, Running)
